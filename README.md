# telejobs

· Un sistema de Registro

· Sistema de CONFIGURACIÓN, con datos tomados por Sesión PHP + AJAX
· Simplificación de toma de datos de Base de Datos (a través de Sesiones PHP y control de UPDATEs en MySQL)
